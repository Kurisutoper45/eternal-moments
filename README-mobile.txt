
Eternal Moments — Chris × Ita (Mobile Fix)
==========================================
Jika musik tidak berputar dan cover tidak tampil di HP:

1) Buka folder ini di aplikasi Spck Editor (Play Store).
   - Tap index.html lalu tekan ikon ▶️ (Run/Preview) di pojok kanan atas.
   - Ini membuat alamat menjadi http://localhost sehingga konten tidak diblokir.

2) Atau buka index.html lewat Chrome sebagai file:// (bukan content://).
   - Dari File Manager, pilih 'Buka dengan' → Chrome.
   - Jika tetap content://, gunakan tombol Share → 'Buka di Chrome' / 'Open in browser'.

3) Pastikan cover berada di: assets/photos/cover.jpg
   - Nama file HARUS 'cover.jpg' huruf kecil.
   - Ganti file ini dengan foto kalian untuk muncul di hero.

4) Musik:
   - File: assets/music/bg-melody.mp3
   - Autoplay: diputar muted saat load, lalu unmute otomatis pada tap/scroll pertama.
